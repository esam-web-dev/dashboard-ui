# Dashboard UI

A modern and responsive dashboard interface built with **React**, **Vite**, and **TailwindCSS**.  
This project provides a professional user experience for managing financial data such as cards, transactions, budgets, subscriptions, loans, savings, and reports.

## 🚀 Features

### 🔹 Cards Overview
- Display card balance
- Masked card number
- Expiration date

### 🔹 Transactions
- Recent transactions list
- Customer name – date – amount

### 🔹 Budget Management
- Cash overview
- Automated and manual expenses
- Category-based budget items

### 🔹 Subscriptions
- Active subscriptions list
- Due dates and amounts

### 🔹 Loans
- Savings target tracking
- Progress indicator
- Remaining amount

### 🔹 Savings
- Savings goals
- Remaining amount
- Date taken

### 🔹 Reports
- Monthly bar chart visualization
- Financial performance overview

### 🔹 Financial Advice
- Section dedicated to financial tips

## 🛠️ Tech Stack

- **React 18**
- **Vite**
- **TailwindCSS**
- **React Router**
- **Context API**
